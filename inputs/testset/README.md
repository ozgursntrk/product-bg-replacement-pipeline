# Test set

Five products photographed on the same table under the same lighting, covering four
difficulty classes for segmentation:

| File | Class | Why it is here |
|---|---|---|
| `01_matte_mug.jpeg` | matte | Baseline; strong product/background contrast |
| `02_matte_book.jpeg` | matte | Flat geometry, easy case |
| `03_glossy_fork.jpeg` | glossy / thin detail | Reflections plus thin tines |
| `04_transparent_glass.jpeg` | transparent | Printed pattern is opaque, the glass is not |
| `05_lowcontrast_headphones.jpeg` | low contrast | Dark object on dark wood, plus a genuine hole in the topology |
